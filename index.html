    About SERPSCOPE

    SERPSCOPE is a fully client-side SEO intelligence platform. It runs in your browser, fetches public pages through a CORS proxy, and computes a multi-dimensional SEO score against five competitors. No data leaves your device except outbound page fetches.

    What it measures

    • On-Page — titles, meta descriptions, heading hierarchy, image alt coverage, internal/external linking, Open Graph & Twitter Cards, canonicals, structured data (JSON-LD).
    • Technical — HTTPS, mobile viewport, robots directives, sitemap discovery, response time, page weight, Core Web Vitals (via PSI), hreflang.
    • Content — word count, Flesch reading ease, keyword density, semantic coverage, heading-to-content ratio, multimedia richness.
    • Off-Page — protocol & security, social presence, domain hints, content freshness signals, third-party trust markers.

    Scoring methodology

    Each signal is normalized 0–100 and weighted by category. The composite SERPSCOPE Index uses: On-Page 30% · Technical 30% · Content 25% · Off-Page 15%. Action priorities (P0–P3) are derived from a signal's impact × ease ratio.

    Limitations

    Off-page authority (backlink graph, referring domains) requires a commercial API such as Ahrefs, Majestic, or Moz. SERPSCOPE estimates off-page health using public signals; for true link metrics, integrate one of those providers via the Settings panel.

    Open source

    Single-page app, MIT-style. Hosted on GitHub Pages. Edit assets/js/analyzer.js to tune scoring weights.
